# Security Policy

## Supported Versions
Main branch and the latest tagged releases receive security attention.

## Reporting a Vulnerability
Create a private security advisory in GitHub Security Advisories for this repository.
Do not open a public issue. Provide steps to reproduce, affected versions, and potential impact.

## Handling
We will acknowledge within 72 hours, triage the report, prepare a fix, and issue a patched release.
We give credit to reporters upon request after a fix is released.
